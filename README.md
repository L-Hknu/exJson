# xls
html5中 可以将xls 转成json 修改重新下载 的功能

关于标题的规范
[] 中括号 代表有路径
() 括号代表表示key值，只解析有key的内容
<> 尖括号代表类型 
基础场景:
{"title" : "阅读数 22", 
"href" : "http://www.google.com", 
 }
|标题（title）|链接（href)|
|-阅读数 2-|-http://www.google.com-|

    | 标题（title  | 链接（href) |
    | ------------- | ------------- |
    | 阅读数 2  | http://www.google.com  |
